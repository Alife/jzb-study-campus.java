Sample Graphs

This directory contains some sample graphs in either YGF (Y Graph Format) or GML (Graph Modeling Language) format.

Both formats are supported by yFiles and thus can be read and written by the yED graph editor application (and with any other application that build upon yFiles).

YGF is the "native" format of yFiles. It is

  • proprietary (-)
  • non-human-readable (-)
  • compact (+)
  • powerful (+)

GML is an open graph interchange format not unlike XML. It is

  • open (+) (see here for specification)
  • human-readable (+)
  • not so compact (-)
  • not so powerful (-)

Note that yFiles also supports an XML-based graph format called GraphML. The yFiles GraphML extension package is available from our Web site.
